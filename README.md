Instalação
Para instalar e executar o projeto, siga os passos abaixo:

# Clone o repositório

git clone https://github.com/seu-usuario/projeto-aleatorio.git

# Acesse a pasta do projeto

cd projeto-aleatorio

# Instale as dependências

npm install

# Inicie o projeto

npm start
